# The Technical Eye

Personal portfolio site - software engineering meets photography.

**Live:** [kartikeybhardwaj.com](https://kartikeybhardwaj.com)

## Features

- Dark minimal design with scroll-triggered animations
- Filterable works section (Engineering / Photography)
- Custom cursor follower
- Mobile-responsive with hamburger menu
- Downloadable resume
- Zero frameworks - vanilla HTML, CSS & JS

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Google Fonts (Space Grotesk, Inter)
- Firebase Hosting

## Run Locally

```bash
# Clone
git clone https://github.com/kartikeybhardwaj/site-resume.git
cd site-resume

# Open
open index.html
```
